# 🚀 CareerVision – AI Career Recommendation System

## 📌 Overview
CareerVision is an AI-powered career recommendation platform that helps students identify suitable career paths based on their skills, interests, education, and achievements. The system provides personalized career suggestions, skill gap analysis, salary estimation, and learning recommendations.

## ✨ Features
- 🎯 AI-Based Career Recommendations
- 🔍 Skill Gap Analysis
- 📚 Course Recommendations
- 📈 Career Readiness Score
- 💰 Salary Estimation (LPA)
- 🗺️ Personalized Learning Path

## 🛠️ Technologies Used
- Python
- Streamlit
- Pandas
- NumPy
- Scikit-Learn
- TF-IDF Vectorization
- Cosine Similarity

## ⚙️ Installation

1. Clone the repository

```bash
git clone https://github.com/jeyakiruhikad/AI_Career_Recommendation_System.git
cd AI_Career_Recommendation_System
```

2. Install dependencies

```bash
pip install -r requirements.txt
```

3. Run the application

```bash
streamlit run careerRecommendation.py
```
